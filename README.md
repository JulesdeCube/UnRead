## 📘 Usage

### Install

```bash
make install # install all dependencies
make build # build Unread
```

### Run

```bash
make run
# or
./build/Unread-[version]
```

where `[version]` is the **UnRead** current version

### Clean

```bash
make clean
```

this will remove all the compilation files
